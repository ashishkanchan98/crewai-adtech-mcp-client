Execution Patterns

Sequential (most crews)
Triage Task → Domain Agent Task → KB Lookup Task → Synthesis Task Each task sees previous output. Good for dependent diagnostics.
Parallel (Use Case 6 — Pre-Launch Audit)
Campaign ──┐ Audience ──┤ Pixel ──┼──▶ GO/NO-GO Deal ──┤ Fraud ──┤ KB ──┘ async_execution=True per task. 6× faster than sequential.
